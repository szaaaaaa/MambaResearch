## v0.4.0 — Dynamic Research OS (2026-03-08 ~ 2026-03-16)

### 架构转型：从固定流程到动态操作系统

这是项目最大的一次架构变更，从静态的多阶段 Agent 流程转变为动态的研究操作系统。

### 新功能
- **Artifact 合约 & Skill Runtime**：正式的数据合约和技能运行时
- **LLM Provider 适配器**：多后端 LLM 支持
- **3-Agent → 6-Agent 架构**：从 3 个角色扩展到 6 个专业角色
- **6-Agent 完整实现**：技能包装器、编排器集成、前后端对接
- **动态路由 UI**：前端展示动态路由决策
- **Dynamic OS Runtime**：全新的运行时引擎，后端完全切换
- **Codex OAuth & MCP Runtime**：外部工具集成后端
- **FAISS 索引后端**：向量检索支持
- **角色专属 LLM 选择**：不同角色可使用不同的 LLM

### 重构
- FastAPI 路由拆分
- 阶段运行时管道更可组合
- 移除遗留 Agent 实现

### 文档
- Dynamic Research OS 中英文技术文档
- 架构规格说明对齐
